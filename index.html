// How it works

Three verbs. Zero servers.

01 / ETCH
Write
it.

Paste text into the app. Pay a small fee in ANT. The content is written directly to the Autonomi Network as immutable, content-addressed storage.

02 / SHARE
Send
it.

You get a permanent address. No login needed on either end. Send it in Slack, a text, an email — the link will still work in ten years.

03 / FETCH
Read
it.

Anyone with the link retrieves the exact content you etched, byte-for-byte. No edits. No takedowns. No expiry. Forever.
